أستطيع مساعدتك في إنشاء موقع CV احترافي! لكن دعني أوضح النقاط التالية:
ما يمكنني فعله:
  • كتابة كود HTML/CSS/JavaScript كامل لموقع CV تفاعلي وجذاب
  • تصميمه بشكل responsive (يعمل على الجوال والكمبيوتر)
  • إضافة أقسام: نبذة عنك، خبرات العمل، المهارات، التعليم، مشاريع، وسائل التواصل
  • إضافة تأثيرات حركية وتصميم عصري
ما لا يمكنني فعله:
  • رفع الموقع على استضافة (hosting) أو شراء دومين
  • تشغيل خادم (server) لاستضافة الموقع
للبدء، أحتاج منك:
  1. معلوماتك الشخصية (الاسم، المسمى الوظيفي، نبذة مختصرة)
  2. خبرات العمل (الشركات، المدة، المهام)
  3. المهارات التقنية واللينة
  4. التعليم والشهادات
  5. مشاريع notable (إن وجدت)
  6. روابط LinkedIn/GitHub/Portfolio
  7. تفضيل التصميم (عصري/كلاسيكي/إبداعي/بسيط)
هل تريد أن تبدأ بإعطائي هذه المعلومات؟ يمكنني إنشاء الموقع كملف HTML واحد جاهز للرفع على أي استضافة مجانية مثل GitHub Pages أو Netlify.

تمام! هنا شرح مفصل لرفع موقعك على GitHub Pages خطوة بخطوة:

الطريقة 1: GitHub Pages (الأفضل للسير الذاتية)

الخطوة 1: أنشئ حساب GitHub

  1. افتح github.com
  2. اضغط Sign Up
  3. اكتب:
    • Username: yousefalqayri (أو أي اسم تبيه)
    • Email: بريدك
    • Password: كلمة مرور
  4. أكمل التحقق (Verify)
  5. اضغط Create Account

الخطوة 2: أنشئ Repository جديد

  1. بعد تسجيل الدخول، اضغط الزر الأخضر + (أعلى اليمين)
  2. اختار New repository
  3. اكتب:
    • Repository name: yousefalqayri.github.io ← مهم جداً يكون بنفس هذا الشكل
    • Description: My Personal CV Website
    • اختر Public ✓
    • اضغط علامة Add a README file
  4. اضغط Create repository

الخطوة 3: ارفع ملف الموقع

  1. داخل الـ Repository، اضغط Add file → Upload files
  2. اسحب ملف Yousef_Alqayri_CV_Updated.html من جهازك
  3. انتظر حتى يكتمل الرفع
  4. في أسفل الصفحة، اكتب:
    • Commit changes: Add CV website
  5. اضغط Commit changes

الخطوة 4: تفعيل GitHub Pages

  1. في الـ Repository، اضغط على Settings (أعلى اليمين)
  2. في القائمة اليسرى، اضغط Pages
  3. في Source، اختار:
    • Branch: main
    • Folder: / (root)
  4. اضغط Save
  5. انتظر 2-5 دقائق
  6. الرابط يظهر في الأعلى: https://yousefalqayri.github.io

الخطوة 5: إعادة تسمية الملف (مهم!)

  1. ارجع للـ Repository
  2. اضغط على اسم الملف Yousef_Alqayri_CV_Updated.html
  3. اضغط القلم ✏️ (Edit)
  4. غير الاسم إلى: index.html
  5. اضغط Commit changes
لماذا index.html؟ لأن GitHub Pages يفتح index.html تلقائياً بدون ما تكتب اسم الملف
